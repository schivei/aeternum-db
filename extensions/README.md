# WASM Extensions

Extensibility system for AeternumDB using WebAssembly.

## License

MIT

## Status

🚧 Under development

## Features (Planned)

- Sandboxed execution environment
- Hot-reload support
- Language-agnostic (any WASM-compatible language)
- Secure ABI for core communication

## Example Extensions

- GraphQL Engine
- Object Layer (OOP)
- Custom indexes
- Procedural languages (Python, JS, .NET Core)
- Integration connectors (Kafka, MQTT)
